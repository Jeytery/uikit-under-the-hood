### CoreAnimation 
- Изначально LayerKit.
- Отвечает за отрисовки всего в iOS приложении. 
- Изначально любое изменеие анимирует. 
- Нужно писать специальный код, чтобы отменить анимацию.  
По сути UIView проксирует все необходимые в CALayer
